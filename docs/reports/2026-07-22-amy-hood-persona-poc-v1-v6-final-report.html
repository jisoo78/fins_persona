01 · Executive Summary / 개요

로컬 RAG 플랫폼은 작동했지만, 인물 복제 가설은 입증되지 않았습니다

핵심 판정: 공개자료→판단 정책→동적 RAG→CFO 조언→행동 평가의 전체 경로는 구현됐습니다. 그러나 일반 모델도 8.20~8.70점과 행동 전환 100%를 기록했고, RAG 증분은 사전 기준 +0.5에 못 미친 +0.31~+0.43이었습니다. 따라서 플랫폼 기술은 Go, Amy Hood 고유 판단 연구는 재평가 조건부 Go, 현재 결과를 근거로 한 복제·제품화 주장은 No-Go입니다.
GO구현 완결성

근거 수집→정책 메모리→검색→응답→채점이 실제로 동작합니다.

FAILRAG 증분 KPI

목표 +0.5 대비 Policy +0.31, Full +0.43을 기록했습니다.

RETEST평가 판별력

재무·회계 전문가와 고난도 문항을 다시 설계해야 합니다.

02 · Background & Problem Definition / 배경과 문제 정의

말투가 아니라 판단 순서와 조건을 재현할 수 있는가

  • 일반 페르소나는 인물의 문체를 흉내 내지만 실제 의사결정 우선순위와 예외를 검증하기 어렵습니다.
  • 공개자료만으로 특정 CFO의 판단 정책을 도출하고 새로운 상황에 적용할 수 있는지가 핵심 기술 질문이었습니다.
  • 외부 상용 API의 보안·통제·마진 의존을 줄이고, 향후 회사 내부에서 통제 가능한 로컬 LLM 서비스 구조를 검증했습니다.
PoC 자원은 Amy Hood의 공개 발언과 사건에서 판단 정책을 추출하고, Main Prompt와 장기 기억 RAG로 CFO 조언을 생성하는 B Track 검증에 집중했습니다.
사용한 로컬 LLM 모델(Gemma 4 E4B·12B)은 회사 인프라에서 운영 가능한 자원 범위를 고려한 선택입니다. 적은 자원으로 상용 API 수준에 접근하는 것이 목표였지만, 이번 PoC가 성능 동등성이나 실제 비용 우위를 입증한 것은 아닙니다.

천장 효과가 발생한 이유

1낮은 실질 난이도

문항에 행동 전환 신호와 정답 방향이 비교적 명시적으로 드러났습니다.

2LLM의 평균 수렴

모델은 불확실한 재무 상황에서 일반적이고 안정적인 CFO 답변으로 수렴했습니다.

3전문가 검토 부재

재무·회계 전문가가 만든 실제 난도의 경계 조건과 함정이 부족했습니다.

다음 평가는 재무·회계 전문가가 실제 의사결정에서 충돌하는 회계 처리, 자본 배분, 현금흐름, 계약 조건과 리스크를 설계하고 정답·오답 근거까지 사전 동결해야 합니다.
로컬 LLM 선택은 단순한 실험비 절감이 아니라 향후 보안·통제·마진을 회사 내부에 두기 위한 제품 전략입니다. 다만 실제 TCO, 동시 처리량, 응답 지연, GPU 요구량과 상용 API 대비 품질은 별도 실험이 필요합니다.

  • 문항: 명시적 전환 단서와 일반 CFO 정답이 많아 모델별 판단 차이가 사라졌습니다.
  • 검증: 재무·회계 전문가와 독립된 인간 평가자가 문항·정답·채점을 검토하지 않았습니다.

높은 점수보다, 믿을 수 있는 판단 기준을 찾는 데 집중했습니다

처음에는 공개자료를 모아 프롬프트와 RAG에 넣으면 의사결정 페르소나를 만들 수 있다고 생각했습니다. 그러나 실제 작업은 훨씬 더 집요한 검증의 연속이었습니다. 프롬프트를 고치고 로컬 모델을 다시 실행하며, 점수가 달라질 때마다 원인이 프롬프트인지, 검색된 기억인지, 모델 변동성인지, Judge의 흔들림인지 하나씩 분리했습니다. 만점에 가까운 결과가 나와도 평가가 쉬웠다면 성공으로 포장하지 않고 문항과 기준을 다시 만들었습니다.

정답이 존재하는가를 의심하게 만든 Costco 시나리오

Costco는 2024년 회원비 인상을 공식 발표했고, CFO는 코로나와 고물가를 고려해 통상적인 인상 시점보다 약 2년 늦췄다고 설명했습니다. 시행 초기에는 회원 반응과 갱신 추세에도 유의미한 변화가 없었습니다. 실제 Costco CFO는 상황이 회복됐다고 보고 미래 전략을 위해 리스크를 감수하고 인상을 실행했지만, 같은 사건에서 착안한 가상 시나리오의 Amy Hood 페르소나는 보류·추가 검토·리스크 완화 쪽으로 주저했습니다.

여기서 근본적인 회의가 생겼습니다. 실제 사건은 결과를 알고 있으므로 성공적인 결정이었다고 평가할 수 있지만, 가상 시나리오의 미래는 누구도 알 수 없습니다. 강행한 Costco CFO와 주저한 Amy Hood 페르소나 중 누가 정답이라고 말할 수 있는지, 사후 결과를 가상의 정답지로 사용하는 것이 타당한지 답하기 어려웠습니다. 결론이 다르다는 이유로 페르소나를 틀렸다고 하면 당시의 불확실성과 위험 선호를 지우고, 반대로 모든 결론을 성향 차이로 인정하면 평가 자체가 성립하지 않습니다. 이 모순은 유명인의 미래 의사결정을 객관식 정답처럼 채점하려는 PoC 접근의 근본적인 한계였습니다.
