07 · Hypothesis Review / 가설 판정

RAG 방향성은 지지됐고, 모델 순위와 행동 전환 판별력은 입증되지 않았습니다

가설관측판정근거
H1. 일반 고성능 LLM도 높은 점수를 기록한다8.20 / 8.70지지Amy 정보 없이 두 모델 모두 8점 이상.
H2. Amy/RAG 조건은 일반 CFO 답변보다 높다+0.82~+1.75기술적 관측점수는 높지만 Judge·반복·Prompt가 달라 인과 효과로 확정 불가.
H3. Policy RAG는 Amy Prompt를 개선한다+0.31방향성 지지동일 Gemma·Judge·5회 조건의 통제 비교.
H4. Full RAG는 Policy RAG를 실질적으로 개선한다+0.12약한 지지추가 입력 비용 대비 효과가 작고 M&A에서는 Policy가 더 높음.
H5. 행동 변화 정합률이 페르소나를 판별한다모두 100%기각일반 외부 모델도 전부 통과해 조건 간 판별력이 없음.
H6. Gemma 모델 자체가 외부 모델보다 우수하다측정 불가입증 불가모델 외 조건이 통제되지 않음.

08 · Validity & Limitations / 유효성 한계

이 보고서의 가장 중요한 표는 점수가 아니라 비교 불가능성의 경계입니다

  1. Judge 불일치: 외부 모델은 Codex가 채점했고 Gemma v6는 로컬 Gemma 4 v2 Judge가 채점했습니다. 동일 점수 척도를 사용했지만 채점자 성향은 통제되지 않았습니다.
  2. 반복 수 불일치: 외부 모델은 각 1회, Gemma는 각 조건 5회입니다. 외부 모델의 평균·분산·선택 안정성은 알 수 없습니다.
  3. 입력 조건 불일치: 외부 모델에는 일반 CFO Prompt, Gemma에는 Amy Prompt와 선택적 RAG가 들어갔습니다. 점수 차이는 모델·Prompt·메모리 효과가 섞인 결과입니다.
  4. 웹 런타임 통제 부족: Gemini·Sonnet의 정확한 샘플링 설정, 숨은 시스템 지침, reasoning budget은 기록되지 않았습니다.
  5. 천장 효과: 외부 모델도 8점 이상이고 Gemma 조건은 9.5점 이상입니다. 문항이 어려운 판단 충돌보다 정답 방향 단서를 많이 제공합니다.
  6. 쌍 노출 위험: A/B 문항의 변화 신호가 명확해, 모델이 Amy 정책 없이도 변화 방향을 쉽게 추론할 수 있습니다.
  7. 모델 명칭: Gemini Flash 3.6과 Claude Sonnet 5 표기는 사용자 제공 실행 정보이며 이 보고서에서 외부적으로 검증하지 않았습니다.

10 · Recommendation / 다음 단계

정식 교차 평가는 같은 Judge·반복·입력 경계로 다시 실행해야 합니다

  1. Gemini와 Sonnet을 각각 최소 3회, 가능하면 Gemma와 동일한 5회로 반복합니다.
  2. 모든 1,050개 후보 답변을 동일한 독립 Judge 하나로 다시 채점합니다.
  3. 일부 표본을 인간 2명 이상이 채점해 LLM Judge와의 일치도를 측정합니다.
  4. A/B 문항을 분리·무작위화하고, 한 대화에서 짝 문항을 함께 노출하지 않습니다.
  5. 정답 방향이 명시된 단어를 줄이고, 두 선택 모두 재무적으로 타당하지만 Amy 우선순위만 다른 시나리오를 추가합니다.
  6. M&A·AI CapEx·주주환원에 “무엇을 포기하고 무엇을 끝까지 보호하는가”를 묻는 경계 문항을 집중 배치합니다.
  7. 외부 모델에는 일반 CFO와 Amy Prompt 두 조건을 모두 적용해 모델 효과와 Prompt 효과를 분리합니다.
